# Morse BLE Keyboard
Keyboard with Morse code input

https://github.com/bem130/MorseBLEkeyboard/assets/79097169/a3413a5b-6844-4ad9-a6f8-b8184e182b64

## HARDWARE
- ESP32 Dev Kit (x1)
- Switch (x2)

### wiring
Connect the two switches to GPIO18,19 and GND respectively
